Chapter 10 — Censored and Truncated Data

This chapter is about data that are incomplete because of observational limits. In astronomy, this happens very often. Instruments have detection limits, and surveys have selection rules.

Censored and truncated data require special treatment. Using standard statistical methods without accounting for these effects can lead to biased results.

Censoring vs. truncation

Censoring and truncation describe two different situations.

Censoring means an object is observed, but its value is not fully known. An upper or lower limit is measured instead of an exact value.

Truncation means objects are completely missing from the dataset if they fall outside some range.

Types of censoring

Right censoring occurs when values above a certain value are unknown. Left censoring occurs when values below a certain value are unknown.

Problems with ignoring censoring

If censored data are treated as exact values or discarded, statistical estimates become biased. Means, correlations, and regression slopes can be strongly distorted, especially when faint objects are ignored.

Survival analysis ideas

Many methods for censored data come from survival analysis. These methods treat limits explicitly instead of ignoring them.

It measures the probability of survival after certain period of time.

The Kaplan–Meier estimator

The Kaplan–Meier estimator is used to estimate distributions with censored data. It builds a stepwise estimate of the cumulative distribution function while accounting for censored observations.

This method allows both detected values and limits to be included.

Regression with censored data

Regression becomes more complicated when the response variable is censored. Ordinary least squares does not apply. Instead, we use likelihood functions.

Maximum likelihood approaches

Maximum likelihood provides a natural framework for censored and truncated data. The likelihood includes contributions from both detected values and limits.

This allows all available information to be used.

Truncated data

In truncated samples, some objects are completely missing. For example, a flux-limited survey only includes objects brighter than a detection threshold. Objects below the threshold never appear in the dataset.
