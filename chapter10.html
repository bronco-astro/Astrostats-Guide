Chapter 10 — Censored and Truncated Data

This chapter is about data that are incomplete because of observational limits. In astronomy, this happens very often. Instruments have detection limits, and surveys have selection rules.

Censored and truncated data require special treatment. Using standard statistical methods without accounting for these effects can lead to biased results.

Why incomplete data appear

Astronomical observations are limited by sensitivity. Some objects are too faint to be detected, while others are only partially measured.

The dataset does not represent the full underlying population. Understanding how data are missing is the first step.

Censoring versus truncation

Censoring and truncation describe different situations.

Censoring means an object is observed, but its value is not fully known. An upper limit is measured instead of an exact value.

Truncation means objects are completely missing from the dataset if they fall outside some range.

This distinction matters because it changes how analysis is done.

Types of censoring

Right censoring occurs when values above a threshold are unknown. Left censoring occurs when values below a threshold are unknown.

Interval censoring occurs when values are only known to lie within a range. Upper limits are a common form of censoring in astronomy.

Problems with ignoring censoring

If censored data are treated as exact values or discarded, statistical estimates become biased.

Means, correlations, and regression slopes can be strongly distorted, especially when faint objects are ignored.

Survival analysis ideas

Many methods for censored data come from survival analysis. These methods treat limits explicitly instead of ignoring them.

Survival analysis was originally developed for time-to-event data but applies naturally to astronomical measurements.

The Kaplan–Meier estimator

The Kaplan–Meier estimator is used to estimate distributions with censored data. It builds a stepwise estimate of the cumulative distribution function while accounting for censored observations.

This method allows both detected values and limits to be included.

Comparing censored samples

Comparing samples that contain censoring requires special tests. Rank-based tests have versions that handle censored data.

Ignoring censoring in comparisons can lead to incorrect conclusions.

Regression with censored data

Regression becomes more complicated when the response variable is censored. Ordinary least squares does not apply directly.

Likelihood-based methods are commonly used instead.

Maximum likelihood approaches

Maximum likelihood provides a natural framework for censored and truncated data. The likelihood includes contributions from both detected values and limits.

This allows all available information to be used.

Truncated data

In truncated samples, some objects are completely missing. For example, a flux-limited survey only includes objects brighter than a detection threshold.

Objects below the threshold never appear in the dataset.

Methods for truncated data

Analysis of truncated data requires modeling the selection process. Likelihood functions are conditioned on objects being observed.

This correction accounts for missing objects and reduces bias.

Astronomical examples

Censored and truncated data appear in luminosity functions, flux measurements, and survey catalogs.

Proper treatment of limits and selection effects is essential for reliable scientific conclusions.

Limits of these methods

These methods rely on assumptions about censoring and truncation mechanisms. If assumptions are wrong, results can still be biased.

Understanding the observational process is essential.
