Chapter 4 — Probability Distributions

Probability distributions describe how values of a measurement are spread. Instead of focusing on a single observation, distributions describe what happens when the same measurement is repeated many times.

In astronomy, distributions are used to model photon counts, measurement errors, detection results, and similar quantities.

A distribution is a model. It does not describe reality exactly, but it captures the main behavior of the data.

Discrete and continuous measurements

Some measurements take only whole-number values, such as the number of photons detected or the result of repeated detection attempts. These are described using discrete probability distributions.

Other measurements vary smoothly over a range of values, such as brightness or distance. These are described using continuous probability distributions.

The type of measurement determines which distribution is appropriate.

The binomial distribution

The binomial distribution is used when a measurement is repeated a fixed number of times, and each repetition has only two possible outcomes.

Examples include detection versus non-detection.

The probability of observing exactly k successes is 

P(k) = ( n choose k ) p^k (1 − p)^(n − k)

( n choose k ) = n! / [ k! (n − k)! ]

The Poisson distribution

The Poisson distribution describes how many times something happens within a fixed interval.

In astronomy, it is often used to model photon counts during an exposure.

The probability of observing k occurrences is 

P(k) = ( λ^k e^(−λ) ) / k!

This distribution is commonly used when occurrences are independent and happen at a constant average rate.

The normal distribution

The normal distribution, also called the Gaussian distribution, is often used to describe measurement errors.

It appears when many small effects contribute to uncertainty.

The probability density function is 

f(x) = (1 / (σ √(2π))) exp[ − (x − μ)² / (2σ²) ]

The normal distribution is symmetric around its center.

Comparing the distributions

The binomial distribution is used when the number of trials is fixed and outcomes are limited to two possibilities.

The Poisson distribution is used when counting how many times something occurs in a fixed interval.

The normal distribution is used for continuous measurements.
